Description: Easily generate QR codes for your posts and pages directly from the WordPress admin interface. This plugin adds a "Generate QR Code" link to your post and page lists, allowing you to quickly create QR codes for sharing URLs.

== Installation ==
1. Upload the `qr-code-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= What does this plugin do? =
It adds a "Generate QR Code" link to each post and page in the admin area, allowing you to generate a QR code for the URL.

= Does it work with custom post types? =
Currently, this plugin only supports posts and pages.
